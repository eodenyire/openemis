# openemis_transportation — Transportation Module

Manages school transport — vehicles, routes, and student allocations.

## Models
- `op.transport` — Transport routes
- `op.transport.vehicle` — Vehicle fleet
- `op.student.transport` — Student transport allocations

## Key Features
- Route and vehicle management
- Student transport allocation per route
- Transport fee integration
- Driver and vehicle tracking

## Depends On
`openemis_core`
