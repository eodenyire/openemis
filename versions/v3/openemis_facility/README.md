# openemis_facility — Facility Management Module

Manages school facilities — classrooms, labs, sports facilities, and equipment.

## Models
- `op.facility` — Facility definitions (name, type, capacity, location)
- `op.facility.booking` — Facility booking/reservation records

## Key Features
- Facility catalog management
- Booking and reservation system
- Availability calendar
- Facility utilization reports

## Depends On
`openemis_core`
