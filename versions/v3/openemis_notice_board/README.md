# openemis_notice_board — Notice Board Module

Manages school announcements and notices.

## Models
- `op.notice.board` — Notice/announcement records (title, content, audience, date)

## Key Features
- Notice creation and publishing
- Audience targeting (all, students, faculty, parents)
- Notice expiry management
- Portal display for students and parents

## Depends On
`openemis_core`
