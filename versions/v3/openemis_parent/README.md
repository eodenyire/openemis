# openemis_parent — Parent Portal Module

Provides a parent-facing portal and manages parent-student relationships.

## Models
- `op.parent` — Parent profiles
- `op.parent.student.rel` — Parent-student relationship mapping

## Key Features
- Parent registration and portal access
- Parent-student linking (one parent can have multiple children)
- Parent portal: view child's attendance, results, fees, timetable
- Parent notifications for absences and fee dues
- Printable parent reports

## Depends On
`openemis_core`
