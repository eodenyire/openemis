# openemis_attendance — Attendance Module

Tracks student and faculty attendance at the class/session level.

## Models
- `op.attendance` — Attendance sheet per class session
- `op.attendance.line` — Individual student attendance record (present/absent/late)
- `op.faculty.attendance` — Faculty attendance records

## Key Features
- Bulk attendance entry per class session
- Attendance reports per student, course, and date range
- Absence notifications
- Attendance percentage calculation
- Wizard for bulk attendance marking
- Printable attendance sheets

## Depends On
`openemis_core`
