# openemis_discipline — Discipline Module

Records and manages student discipline incidents.

## Models
- `op.discipline` — Discipline incident records (student, type, date, action taken)
- `op.discipline.type` — Incident categories

## Key Features
- Incident recording and tracking
- Disciplinary action management
- Parent notification on incidents
- Discipline reports

## Depends On
`openemis_core`
