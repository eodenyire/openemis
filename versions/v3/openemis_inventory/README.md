# openemis_inventory — Inventory Module

Manages school inventory — stationery, equipment, and consumables.

## Models
- `op.inventory.item` — Inventory items
- `op.inventory.transaction` — Stock in/out transactions

## Key Features
- Item catalog with categories
- Stock tracking (in, out, balance)
- Low stock alerts
- Inventory reports

## Depends On
`openemis_core`
