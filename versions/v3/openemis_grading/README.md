# openemis_grading — Grading Module

Provides configurable grading systems and GPA calculation.

## Models
- `op.grading.system` — Grading system definitions
- `op.grading.line` — Grade boundaries (A: 80-100, B: 60-79, etc.)

## Key Features
- Multiple grading systems per institution
- Configurable grade boundaries and GPA points
- Integration with exam results

## Depends On
`openemis_core`, `openemis_exam`
