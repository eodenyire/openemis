# openemis_erp — ERP Meta-Module

A meta-module that bundles all core openEMIS modules into a single installable package. Installing this module installs the complete openEMIS suite.

## What It Installs

Declares dependencies on all openEMIS modules so that installing `openemis_erp` from the Odoo Apps menu triggers installation of the entire suite in the correct dependency order.

## Usage

In the Odoo Apps menu, search for **openEMIS ERP** and click Install. All modules will be installed automatically.

## Depends On
All openEMIS modules.
