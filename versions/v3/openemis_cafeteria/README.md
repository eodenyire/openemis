# openemis_cafeteria — Cafeteria Module

Manages school cafeteria operations — menu, orders, and billing.

## Models
- `op.cafeteria.menu` — Menu items with pricing
- `op.cafeteria.order` — Student orders

## Key Features
- Daily menu management
- Student order tracking
- Cafeteria billing integration

## Depends On
`openemis_core`
