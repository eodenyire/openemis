# openemis_scholarship — Scholarship Module

Manages student scholarships and bursaries.

## Models
- `op.scholarship` — Scholarship definitions (name, amount, criteria)
- `op.scholarship.application` — Student scholarship applications

## Key Features
- Scholarship catalog management
- Application and approval workflow
- Fee waiver integration
- Scholarship reports

## Depends On
`openemis_core`, `openemis_fees`
