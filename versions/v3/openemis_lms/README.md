# openemis_lms — Learning Management System Module

Provides a digital learning platform for course content delivery, assignments, and online assessments.

## Models
- `op.lms.course` — LMS course definitions
- `op.lms.content` — Course content items (videos, documents, links)
- `op.lms.enrollment` — Student enrollments
- `op.lms.quiz` — Online quizzes
- `op.lms.quiz.question` — Quiz questions
- `op.lms.quiz.result` — Student quiz results

## Key Features
- Course content management (upload videos, PDFs, links)
- Student enrollment and progress tracking
- Online quizzes with auto-grading
- Discussion forums per course
- Integration with openemis_core student records

## Depends On
`openemis_core`
