# openemis_blog — Educational Blog Module

A publishing platform for mentors, teachers, and professionals to share educational articles accessible to all students.

## Models
- `op.blog.post` — Blog articles (title, content, author, category, grade-level tags)
- `op.blog.category` — Blog categories
- `op.blog.comment` — Student comments on posts

## Key Features
- Article creation and publishing by approved authors (mentors, teachers, professionals)
- Category management
- Grade-level tagging for targeted content
- Comment moderation
- Student-facing blog portal

## Depends On
`openemis_core`
