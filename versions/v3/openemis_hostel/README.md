# openemis_hostel — Hostel Module

Manages student hostel accommodation — rooms, allocations, and fees.

## Models
- `op.hostel` — Hostel buildings
- `op.hostel.room` — Rooms with capacity and type
- `op.hostel.room.allocation` — Student room allocations

## Key Features
- Hostel and room management
- Room allocation per student and academic year
- Occupancy tracking
- Hostel fee integration

## Depends On
`openemis_core`
