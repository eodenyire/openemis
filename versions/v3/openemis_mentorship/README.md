# openemis_mentorship — Mentorship Platform Module

Connects students with approved mentors (professionals, teachers, parents) through direct messages and group discussion forums.

## Models
- `op.mentor` — Mentor profiles and approval status
- `op.mentorship.session` — Mentorship session records
- `op.mentorship.message` — Direct messages between students and mentors
- `op.mentorship.forum` — Group discussion forums
- `op.mentorship.forum.post` — Forum posts and replies

## Key Features
- Mentor registration and admin approval workflow
- Student-mentor matching
- Direct messaging (DMs)
- Group discussion forums per topic/subject
- AI-assisted question routing to the most relevant mentor
- Mentor activity tracking

## Depends On
`openemis_core`
