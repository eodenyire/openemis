# openemis_assignment — Assignment Module

Manages assignment creation, student submissions, and grading.

## Models
- `op.assignment` — Assignment definitions (title, subject, due date, marks)
- `op.assignment.submission` — Student submissions

## Key Features
- Assignment creation per subject and batch
- Student submission tracking
- Marks entry and feedback
- Submission deadline management
- Assignment reports

## Depends On
`openemis_core`
